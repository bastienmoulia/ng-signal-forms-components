Date & Time Input Components

Date and time input components provide various ways to select dates, times, and combinations thereof.

Input Date

Date picker for selecting a specific date.

Live Demo

Value: {{ model().date }}

Usage

<ngdsf-input-date
  [field]="() => form.date()"
  [params]="{
    label: 'Select Date',
    min: '2024-01-01',
    max: '2025-12-31'
  }"
/>

Input Time

Time picker for selecting a specific time of day.

Live Demo

Value: {{ model().time }}

Usage

<ngdsf-input-time
  [field]="() => form.time()"
  [params]="{
    label: 'Select Time',
    min: '09:00',
    max: '17:00',
    step: 900
  }"
/>

Common Parameters

Property Type Description
label string Field label text
placeholder string Placeholder text
min string Minimum allowed value
max string Maximum allowed value
step number Step increment (for time: seconds)
disabled boolean Disable the field
readonly boolean Make field read-only
