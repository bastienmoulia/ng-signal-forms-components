Other Input Components

Additional specialized input components.

Input Color

Color picker for selecting colors.

Live Demo

Selected Color: {{ model().color }}

Usage

<ngdsf-input-color
  [field]="() => form.color()"
  [params]="{
    label: 'Choose Color'
  }"
/>

Input File

File upload input.

Live Demo

Selected: {{ model().file?.name || 'No file selected' }}

Usage

<ngdsf-input-file
  [field]="() => form.file()"
  [params]="{
    label: 'Upload File',
    accept: 'image/*',
    multiple: true
  }"
/>

Parameters

Property Type Description
accept string File types to accept (e.g., 'image/*', '.pdf')
multiple boolean Allow multiple file selection
