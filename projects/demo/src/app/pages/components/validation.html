Validation Components

Components for displaying validation errors in your forms.

Field Errors

Display validation errors for individual fields.

Live Demo

Usage

<ngdsf-input-text
  [field]="() => form.username()"
  [params]="{ label: 'Username' }"
/>
<ngdsf-field-errors [field]="() => form.username()" />

Form Errors

Display all validation errors for the entire form.

Live Demo

Usage

<form>
  <!-- form fields -->
  <ngdsf-form-errors [form]="myForm" />
</form>

Validation Setup

Set up validations using Angular's signals-based forms API:

import { form, required, email, minLength } from '@angular/forms/signals';

model = signal({
  username: '',
  email: ''
});

myForm = form(this.model, (p) => {
  required(p.username, { message: 'Username is required' });
  minLength(p.username, 3, { message: 'Min 3 characters' });
  required(p.email, { message: 'Email is required' });
  email(p.email, { message: 'Invalid email' });
});
