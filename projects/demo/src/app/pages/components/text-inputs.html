Text Input Components

Text input components handle various types of text-based user input with built-in validation and formatting.

Input Text

Standard single-line text input field.

Live Demo

Value: {{ model().name }}

Usage

<ngdsf-input-text
  [field]="() => form.name()"
  [params]="{
    label: 'Name',
    placeholder: 'Enter your name',
    minlength: 2,
    maxlength: 50
  }"
/>

Parameters

Property Type Description
label string Field label text
placeholder string Placeholder text
minlength number Minimum length
maxlength number Maximum length
pattern string Validation pattern (regex)
autocomplete string HTML autocomplete attribute
disabled boolean Disable the field
readonly boolean Make field read-only
className string Custom CSS class

Textarea

Multi-line text input for longer content.

Live Demo

Value: {{ model().message }}
Length: {{ model().message.length }} / 200

Usage

<ngdsf-textarea
  [field]="() => form.message()"
  [params]="{
    label: 'Message',
    placeholder: 'Enter your message...',
    rows: 4,
    cols: 50,
    maxlength: 200
  }"
/>

Additional Parameters

Property Type Description
rows number Number of visible text rows
cols number Number of visible text columns
wrap string Text wrapping mode ('hard' | 'soft')
