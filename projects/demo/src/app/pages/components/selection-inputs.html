Selection Input Components

Components for selecting from predefined options.

Select (Dropdown)

Dropdown menu for selecting from a list of options.

Live Demo

Selected: {{ model().country }}

Usage

<ngdsf-select
  [field]="() => form.country()"
  [params]="{
    label: 'Country',
    options: [
      { value: '', label: 'Select...' },
      { value: 'us', label: 'United States' },
      { value: 'ca', label: 'Canada' }
    ]
  }"
/>

Option Types

Type Properties Description
option value, label, disabled Single option
optgroup label, options[], disabled Grouped options
hr - Visual separator
