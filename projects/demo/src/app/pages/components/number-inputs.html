Number Input Components

Number input components handle numeric values with optional min/max constraints and step values.

Input Number

Standard numeric input field with increment/decrement controls.

Live Demo

{{ model() | json }}

Usage

<ngdsf-input-number
  [field]="() => form.age()"
  [params]="{
    label: 'Age',
    min: 0,
    max: 120,
    step: 1
  }"
/>

Parameters

Property Type Description
label string Field label text
placeholder string Placeholder text
min number Minimum value
max number Maximum value
step number Increment/decrement step
disabled boolean Disable the field
readonly boolean Make field read-only

Input Range

Slider control for selecting a numeric value within a range.

Usage

<ngdsf-input-range
  [field]="() => form.volume()"
  [params]="{
    label: 'Volume',
    min: 0,
    max: 100,
    step: 1
  }"
/>

Parameters

Property Type Description
label string Field label text
min number Minimum value (required)
max number Maximum value (required)
step number Step increment
disabled boolean Disable the slider
