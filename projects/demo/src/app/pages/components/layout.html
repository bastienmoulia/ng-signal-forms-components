Layout Components

Components for organizing and structuring forms.

Field Group

Group related fields together visually.

Usage

import { NgdsfFieldGroup } from 'ng-dynamic-signal-form';

<ngdsf-field-group [title]="'Personal Information'">
  <ngdsf-input-text ... />
  <ngdsf-input-email ... />
</ngdsf-field-group>

Tabs

Organize form sections into tabs.

Usage

import { NgdsfTabs, NgdsfTabPanel } from 'ng-dynamic-signal-form';

<ngdsf-tabs>
  <ngdsf-tab-panel [title]="'Basic Info'">
    <!-- Form fields -->
  </ngdsf-tab-panel>
  <ngdsf-tab-panel [title]="'Address'">
    <!-- Form fields -->
  </ngdsf-tab-panel>
</ngdsf-tabs>

Field Template

The base template wrapper used by all field components. Provides consistent structure with label, error display, and content projection.

Usage

import { NgdsfFieldTemplate } from 'ng-dynamic-signal-form';

<ngdsf-field-template
  [field]="() => form.myField()"
  [params]="{ label: 'My Field' }"
>
  <!-- Custom input element -->
  <input type="text" ... />
</ngdsf-field-template>
