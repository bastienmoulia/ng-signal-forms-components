Field Parameters Reference

Complete reference of all available field parameters and their types.

Base Field Parameters (NgdsfFieldParams)

All field components inherit these common parameters:

Parameter Type Description Default
label string Field label text -
placeholder string Placeholder text -
disabled boolean Whether field is disabled false
readonly boolean Whether field is read-only false
className string Custom CSS class name(s) -

Text Input Parameters

Additional parameters for text-based inputs (InputText, InputEmail, InputPassword, etc.):

Parameter Type Description
minlength number Minimum character length
maxlength number Maximum character length
pattern string Validation regex pattern
autocomplete string HTML autocomplete attribute
size number Visible width in characters

Number Input Parameters

Additional parameters for InputNumber and InputRange:

Parameter Type Description
min number Minimum value
max number Maximum value
step number Increment/decrement step

Date/Time Input Parameters

Additional parameters for date and time inputs:

Parameter Type Description
min string Minimum date/time value
max string Maximum date/time value
step number Step increment (in seconds for time)

Select Parameters

Parameters specific to NgdsfSelect:

Parameter Type Description
options NgdsfSelectItem[] Array of options, optgroups, or separators
multiple boolean Allow multiple selection
size number Number of visible options

Select Option Types

// Simple option
{ value: 'us', label: 'United States' }

// Disabled option
{ value: 'es', label: 'Spain', disabled: true }

// Option group
{
  type: 'optgroup',
  label: 'Europe',
  options: [
    { value: 'uk', label: 'United Kingdom' },
    { value: 'fr', label: 'France' }
  ]
}

// Horizontal separator
{ type: 'hr' }

Textarea Parameters

Parameters specific to NgdsfTextarea:

Parameter Type Description
rows number Number of visible text rows
cols number Number of visible text columns
wrap string Text wrapping ('hard' | 'soft')
minlength number Minimum character length
maxlength number Maximum character length

File Input Parameters

Parameters specific to NgdsfInputFile:

Parameter Type Description
accept string Accepted file types (e.g., 'image/*', '.pdf')
multiple boolean Allow multiple file selection

Radio Button Parameters

Parameters specific to NgdsfInputRadio:

Parameter Type Description
value string | number The value this radio button represents
