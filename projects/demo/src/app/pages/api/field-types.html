Field Types Reference

Complete reference of all available field types in the NgdsfFieldType enum.

NgdsfFieldType Enum

The NgdsfFieldType enum defines all available field types:

Enum Value Component HTML Input Type Description
InputText NgdsfInputText text Single-line text input
InputEmail NgdsfInputEmail email Email address input
InputPassword NgdsfInputPassword password Password input (masked)
InputSearch NgdsfInputSearch search Search input
InputTel NgdsfInputTel tel Telephone number input
InputUrl NgdsfInputUrl url URL input
InputNumber NgdsfInputNumber number Numeric input
InputRange NgdsfInputRange range Slider input
InputDate NgdsfInputDate date Date picker
InputTime NgdsfInputTime time Time picker
InputDatetimeLocal NgdsfInputDatetimeLocal datetime-local Date and time picker
InputMonth NgdsfInputMonth month Month and year picker
InputWeek NgdsfInputWeek week Week and year picker
InputCheckbox NgdsfInputCheckbox checkbox Checkbox input
InputRadio NgdsfInputRadio radio Radio button input
InputColor NgdsfInputColor color Color picker
InputFile NgdsfInputFile file File upload
InputHidden NgdsfInputHidden hidden Hidden input
Select NgdsfSelect - Dropdown selection
Textarea NgdsfTextarea - Multi-line text input

Usage with NgdsfFields

When using the NgdsfFields component, specify the field type in your params object:

import { NgdsfFieldType, NgdsfFormParams } from 'ng-dynamic-signal-form';

formParams: NgdsfFormParams = {
  username: {
    type: NgdsfFieldType.InputText,
    label: 'Username'
  },
  email: {
    type: NgdsfFieldType.InputEmail,
    label: 'Email'
  },
  birthdate: {
    type: NgdsfFieldType.InputDate,
    label: 'Birth Date'
  }
};

Direct Component Usage

You can also import and use components directly without NgdsfFields:

import { NgdsfInputText, NgdsfInputEmail } from 'ng-dynamic-signal-form';

@Component({
  imports: [NgdsfInputText, NgdsfInputEmail],
  template: `
    <ngdsf-input-text [field]="() => form.username()" />
    <ngdsf-input-email [field]="() => form.email()" />
  `
})
