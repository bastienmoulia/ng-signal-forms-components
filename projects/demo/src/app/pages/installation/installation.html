Installation

Prerequisites

Before installing, ensure you have:

  • Angular v21 or higher
  • Node.js v18 or higher
  • npm or yarn package manager

From Source (Current)

The package is not yet published to npm. To use it in your project, you can install from the source repository:

Step 1: Clone the Repository

git clone https://github.com/bastienmoulia/ng-signal-forms-components.git
cd ng-signal-forms-components

Step 2: Install Dependencies

npm install --ignore-scripts

Step 3: Build the Library

ng build ng-dynamic-signal-form

Step 4: Link to Your Project

You can use the built library in several ways:

Option A: Copy the dist folder

cp -r dist/ng-dynamic-signal-form /path/to/your/project/node_modules/

Option B: Use npm link

# In the ng-signal-forms-components directory
cd dist/ng-dynamic-signal-form
npm link

# In your project directory
npm link ng-dynamic-signal-form

Option C: Use a workspace

If you're building an Angular workspace, you can add this library as a project and reference it directly.

From npm (Future)

Once published to npm, installation will be straightforward:

npm install ng-dynamic-signal-form

Or with yarn:

yarn add ng-dynamic-signal-form

Usage in Your Project

After installation, import components in your Angular components:

import { Component, signal } from '@angular/core';
import { form, required } from '@angular/forms/signals';
import { 
  NgdsfFields, 
  NgdsfFieldType, 
  NgdsfFormParams 
} from 'ng-dynamic-signal-form';

@Component({
  selector: 'app-my-form',
  imports: [NgdsfFields],
  template: `
    <form>
      <ngdsf-fields [params]="params" [form]="myForm" />
    </form>
  `,
})
export class MyFormComponent {
  model = signal({ username: '' });
  
  params: NgdsfFormParams = {
    username: { 
      type: NgdsfFieldType.InputText, 
      label: 'Username' 
    }
  };
  
  myForm = form(this.model, (p) => {
    required(p.username);
  });
}

Importing Individual Components

You can also import individual field components directly:

import { NgdsfInputText } from 'ng-dynamic-signal-form';
import { NgdsfInputEmail } from 'ng-dynamic-signal-form';
import { NgdsfSelect } from 'ng-dynamic-signal-form';

@Component({
  selector: 'app-custom-form',
  imports: [NgdsfInputText, NgdsfInputEmail, NgdsfSelect],
  template: `
    <ngdsf-input-text [field]="() => form.name()" [params]="nameParams" />
    <ngdsf-input-email [field]="() => form.email()" [params]="emailParams" />
  `,
})
export class CustomFormComponent {
  // ... component implementation
}

Troubleshooting

Build Errors

If you encounter build errors, ensure that:

  • You're using Angular v21 or higher
  • TypeScript version is 5.9 or higher
  • All peer dependencies are installed

Import Errors

If you see "Cannot find module" errors:

  • Verify the library is built: check that dist/ng-dynamic-signal-form exists
  • Clear your node_modules cache: rm -rf node_modules && npm install
  • Restart your development server
